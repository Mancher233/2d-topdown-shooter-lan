# SETUP.md -- 游戏安装与运行指南

> 本指南面向完全没有编程经验的同学，请按步骤操作。
> 同时提供了 **Mac** 和 **Windows** 两个系统的说明。

---

## 第一步：安装 Node.js

Node.js 是运行我们游戏服务器的环境。

1. 打开浏览器，访问 https://nodejs.org/
2. 下载 **LTS（长期支持）版本**（左边那个绿色按钮）
3. 双击下载的安装包，一路点 "Next" 安装
4. 安装完成后，打开“终端”（命令行）验证安装：
   - **Mac**: 按 `Cmd + 空格`，输入 `terminal`，回车
   - **Windows**: 按 `Win + R`，输入 `cmd`，回车；或者在开始菜单搜索 "PowerShell"
5. 在终端中输入：
   ```
   node --version
   ```
   如果显示类似 `v20.x.x` 的版本号，说明安装成功。

> **Windows 用户注意：** 如果提示“node 不是内部或外部命令”，说明安装时没有勾选 "Add to PATH"。请重新运行安装程序，确保勾选该选项。

---

## 第二步：准备项目文件

1. 把下载的 zip 文件解压到一个文件夹
2. 解压后应该能**直接看到** `index.html` 文件（这是提交平台的要求）
3. 文件夹结构应该是这样的：
   ```
   你的文件夹/
     index.html         ← 游戏入口页面
     css/               ← 样式文件
     js/                ← 游戏代码
     server/            ← 服务器代码
     SETUP.md           ← 你正在看的这个文件
     README.md          ← 项目总览与玩法说明
     ATTRIBUTIONS.md    ← 声明与致谢
   ```

---

## 第三步：安装服务器依赖

1. 打开终端
2. 用 `cd` 命令进入项目的 `server` 文件夹：
   - **Mac 示例**（文件夹在桌面）：
     ```
     cd ~/Desktop/你的文件夹名/server
     ```
   - **Windows 示例**（文件夹在桌面）：
     ```
     cd C:\Users\你的用户名\Desktop\你的文件夹名\server
     ```
     或者在文件资源管理器中进入 `server` 文件夹，然后在地址栏输入 `cmd` 回车（会自动在该目录打开终端）
3. 运行以下命令安装 WebSocket 库：
   ```
   npm install
   ```
   这会安装一个叫 `ws` 的小库（WebSocket 通信用的）。安装完成后会看到 `node_modules` 文件夹。

---

## 第四步：启动服务器

1. 确保你还在 `server` 文件夹中
2. 运行：
   ```
   node server.js
   ```
3. 你应该看到：
   ```
   ========================================
     2D Top-Down Shooter Server Running!
     Open http://localhost:3000
   ========================================
   ```
4. **不要关闭终端！** 服务器需要一直运行。

---

## 第五步：在浏览器中打开游戏

1. 打开浏览器（Chrome / Firefox / Edge 都可以）
2. 地址栏输入：`http://localhost:3000`
3. 你应该看到游戏的大厅界面，有"创建游戏"和"加入游戏"两个按钮

---

## 第六步：局域网双人对战

### 在同一台电脑上测试（单人测试）

1. 在第一个浏览器标签页中打开 `http://localhost:3000`，点击“创建游戏”
2. 新开一个标签页，同样打开 `http://localhost:3000`，输入 `localhost`，点击“加入游戏”
3. 两个标签页都应该进入游戏画面

> **提示：** 同一台电脑上测试时，只能操作当前激活的标签页。切换到哪个标签页，就能操作哪个玩家。另一个玩家会停在原地不动（但不会死）。这是正常的——真正的双人对战需要在两台设备上测试。

### 在两台设备上对战（真正的多人模式）

**设备 A（房主）：**
1. 按上面步骤 1-4 启动服务器
2. 在浏览器打开 `http://localhost:3000`
3. 点击"创建游戏"

**设备 B（加入者）：**
1. 确保设备 B 和设备 A 连接在**同一个 Wi-Fi** 网络
2. 需要知道设备 A 的局域网 IP 地址：
   - **Mac**: 在设备 A 的终端输入 `ifconfig | grep "inet "`，找到类似 `192.168.x.x` 的地址
   - **Windows**: 在设备 A 的终端（CMD）输入 `ipconfig`，找到 "无线局域网适配器" 下的 "IPv4 地址"（通常是 `192.168.x.x` 格式）
3. 在设备 B 的浏览器中打开 `http://设备A的IP:3000`
   - 例如：`http://192.168.1.100:3000`
4. 输入设备 A 的 IP 地址，点击"加入游戏"

***注：UCAS校园网暂无法实现联机功能，请使用手机热点等。***

## 常见问题

**Q: 浏览器显示“无法连接”？**
- 确保两台设备在同一个 Wi-Fi 网络
- 确保服务器终端没有关闭
- **Mac**: 在“系统设置 > 网络 > 防火墙”中确保没有阻止传入连接
- **Windows**: 在“Windows Defender 防火墙”中允许 Node.js 通过防火墙。
  操作步骤：控制面板 → Windows Defender 防火墙 → 允许应用通过防火墙 → 点击“更改设置” → 找到 `Node.js` 并勾选“专用”和“公用”
  如果列表中没有 Node.js，点击“允许其他应用”并找到 `node.exe`（通常在 `C:\Program Files\nodejs\node.exe`）

**Q: npm install 报错？**
- 确保 Node.js 已正确安装（`node --version` 有输出）
- 尝试关闭终端重新打开
- **Windows**: 如果权限不足，尝试以管理员身份运行 CMD（右键 CMD → “以管理员身份运行”）

**Q: 画面不显示？**
- 按 F12 打开浏览器开发者工具，查看 "Console" 标签中的错误信息
